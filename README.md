# Base Express & Mongo

This is basic express and mongodb configuration for most of my project

### Technology

- Express
- Typescript
- Mongoose

### Code Style

- Eslint Airbnb
- Prettier
- Editorconfig

### Testing

- Jest
- Supertest

### Run Project

Install

```sh
$ npm install
```

Development

```sh
$ npm run dev
```

Build

```sh
$ npm run build
```

Production

```sh
$ npm run prod
```
